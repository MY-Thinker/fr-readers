# 丰润区 2026 年第二届 "书香沁夏・韵满丰润" 全域阅读活动 - 产品需求文档

## 1. 产品概述

为丰润区 2026 年第二届 "书香沁夏・韵满丰润" 全域阅读活动打造一款手机端活动展示页面。页面以曹雪芹文化为精神内核，融入曹雪芹公园、曹雪芹诗文及本土读书名人素材，呈现一城书韵的文化氛围。副标题 "一城书韵承夏序 万卷风华启新程" 贯穿视觉主线。

- 目标用户：丰润区居民、阅读爱好者、文化志愿者、中小学生
- 价值：以文化名人 + 全民阅读为载体，推动书香丰润建设，让阅读成为夏日最美的城市风景

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 普通访客 | 无需注册 | 浏览全部页面内容、点击跳转简道云、观看专家视频 |
| 管理员 | 后台登录 | 在专家讲座页后台上传优质专家视频（预留后台入口） |

### 2.2 功能模块

1. **首页（活动主页）**：顶部通栏轮播 + 活动主标题与副标题 + 活动概述 + 曹雪芹文化专栏 + 本土读书名人 + 活动参与入口
2. **专家阅读指导讲座详情**：讲座列表、嘉宾介绍、时间地点，**预留视频上传窗口**（管理员后台可上传优质专家视频，前端展示视频播放区）
3. **活动任务详情**：**预留 6 个简道云二级跳转窗口**，每个窗口对应一个简道云任务表单链接（链接地址预留，后续填充）
4. **心得展示墙**：与简道云表单数据适配，**当前阶段先留出适配区与占位**，后续接入简道云数据源

| 首页 | 活动主题区 | 主标题 "书香沁夏・韵满丰润"、副标题 "一城书韵承夏序 万卷风华启新程"、活动时间与主办单位 |

## 3. 核心流程

### 3.1 普通访客流程

用户进入首页 → 浏览顶部通栏轮播（三屏切换）→ 点击任一通栏进入对应详情页 → 浏览详情 → 返回首页继续了解曹雪芹文化、本土读书名人 → 完成参与

### 3.2 管理员流程

管理员登录后台 → 进入专家讲座管理 → 上传专家视频 → 视频在前端讲座页播放区展示

### 3.3 简道云跳转流程

用户在活动任务页 → 点击 6 个任务卡片之一 → 二级跳转到对应简道云表单 → 完成任务填报

## 4. 用户界面设计

### 4.1 设计风格

**红楼雅韵 · 暮夏书卷**：以曹雪芹《红楼梦》古典美学为灵感，融合水墨意境与现代移动端排版。

- 主色：暮霞红 `#9B2D20`（红楼朱漆）、宣纸米 `#F5EFE0`（古卷底色）
- 辅色：青玉绿 `#3E5C50`（夏意沁绿）、墨黛黑 `#1F1A17`（水墨笔痕）、金箔黄 `#C8A96A`（书卷金边）
- 按钮风格：仿古印章方形圆角，朱砂红底配米色描边
- 字体：标题用思源宋体 Heavy（衬线、有书卷气）；正文用思源宋体 Regular；点缀用 Ma Shan Zheng（书法手写体）
- 布局：移动端单列卡片流，顶部固定通栏轮播
- 装饰元素：印章纹、卷轴边、水墨晕染、竹叶纹、回字纹边框
- 动效：轮播自动横滑 + 淡入淡出；卡片浮起；页面滚动渐显

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 首页 | 顶部通栏轮播 | 全宽 16:9 轮播图，三屏（专家讲座/活动任务/心得展示），底部圆点指示器，左右轻扫切换 |
| 首页 | 主标题区 | 竖排印章 + 主标题宋体大字 + 副标题小字 + 水墨晕染背景 |
| 首页 | 曹雪芹文化专栏 | 卷轴样式卡片，含曹雪芹公园图、诗文节选、印章落款 |
| 首页 | 本土读书名人 | 头像 + 姓名 + 推荐书目卡片 |
| 首页 | 活动入口 | 三个朱砂印章风格按钮 |
| 专家讲座页 | 视频上传窗口（预留） | 视频播放区 + 上传按钮占位 + 视频列表占位 |
| 活动任务页 | 简道云跳转窗口 ×6 | 6 个任务卡片，每个含任务名 + 简述 + 跳转图标，点击外链跳转 |
| 心得展示页 | 简道云适配区（预留） | 适配简道云字段的卡片骨架，空数据占位提示 |
| 详情页 | 通用 | 顶部返回栏 + 内容卡片流 + 底部装饰 |

### 4.3 响应式

- 移动端优先设计（mobile-first），基础宽度 375px，最大内容宽度 480px 居中
- 触摸优化：轮播支持左右滑动手势，按钮热区 ≥ 44px
- 桌面端：内容居中，最大宽度 480px，左右留白带水墨纹理装饰

### 4.4 3D 场景

不适用。
